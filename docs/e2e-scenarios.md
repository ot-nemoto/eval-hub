# e2e-scenarios.md — E2Eテストシナリオ

PlaywrightMCP が E2E テストを実施するためのシナリオ集。

---

## 報告フォーマット

各シナリオの結果を以下のステータスで報告する。

| ステータス | 意味 |
|----------|------|
| OK | 期待値通り |
| NG | 期待値と異なる（再現手順と実際の挙動を記載） |
| SKIP | 技術的制約等で実行不可（理由を記載） |

# 外部 REST API

## 現状

eval-hub は現時点で外部向けの REST API エンドポイントを提供していない。

書き込み操作はすべて **Server Actions**（`actions.ts`）で実装している。Server Actions の仕様は [`docs/actions.md`](actions.md) を参照。

---

## 将来の外部 API 追加時

外部連携（API キー認証・Webhook 等）が必要になった場合、以下の形式でエンドポイントを定義する。

| セクション | 内容 |
|-----------|------|
| エンドポイント一覧 | メソッド・パス・概要 |
| リクエスト定義 | ヘッダー・ボディのスキーマ |
| レスポンス定義 | 正常系・エラー系のスキーマ |
| エラーレスポンス | ステータスコード・エラーコード一覧 |
